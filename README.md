# Rock Paper Scissor Game 🎮

A fun and interactive Rock Paper Scissor game built with **React** and **Vite**.

## Features ✨

- 🎯 Play against the computer AI
- 📊 Track scores and win streaks
- 📜 View game history (last 5 games)
- 🎨 Beautiful gradient UI with animations
- 🔄 Reset game button to start fresh
- 📱 Responsive design

## How to Play 🎯

1. Click on Rock 🪨, Paper 📃, or Scissor ✂️
2. The computer makes a random move
3. Compare moves and see who wins
4. Track your score and win streak
5. Click "Reset Game" to start over



## Technologies Used 🛠️

- React (useState hooks)
- Vite
- CSS3 (Gradients & Animations)
- JavaScript

## Game Logic 🧠

- Rock beats Scissor
- Scissor beats Paper
- Paper beats Rock
- Same moves = Tie
